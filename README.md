## Get Random Quote

```HTTP
GET /quote
```
```
https://quotenjoke.herokuapp.com/quote
```

Returns a single random quote

## Get Random Joke

```HTTP
GET /joke
```
```
https://quotenjoke.herokuapp.com/joke
```

Returns a single random joke
